# Hybrid-Edge-Cloud-Fault-Detection
AI-based Industrial Fault Detection System using Raspberry Pi, MQTT, LSTM and Grafana for real-time monitoring and predictive maintenance.
## Technologies Used
- Raspberry Pi
- Python
- MQTT Protocol
- Grafana
- InfluxDB
- TensorFlow
- LSTM Autoencoder
- Isolation Forest
- SHAP Explainability
## Features
- Real-time sensor monitoring
- Fault detection using AI
- MQTT cloud communication
- Grafana dashboard visualization
- Explainable AI using SHAP
- Edge-cloud architecture
## Hardware Components
- Raspberry Pi
- MPU6050 Sensor
- DHT22 Sensor
- ACS712 Current Sensor
- Relay Module
